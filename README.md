# React Frontend Test

## Getting Started
To run the app you need `node`,`npm` and `react.js`

1. cd into project folder
2. run 
    ``` cmd
    npm install
    npm start
    ```
    to install the dependency and start the project

## Use Package
* Axios
* Bootstrap
* react
* react-dom
* uuid
